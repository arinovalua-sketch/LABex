# Software Engineering Labs Portfolio

## Overview

This repository contains UML diagrams and system design tasks completed as part of the Software Engineering course.

Each lab demonstrates different modeling techniques and real-world system representations.

---

## Lab 2 – Use Case Diagram

A UML Use Case Diagram for the Lost & Found Portal system.

* Actors: User, Admin
* Demonstrates system interactions
* Includes <<include>> and <<extend>> relationships

📂 Folder: `Lab2_UseCase_Diagrams`

---

## Lab 3 – Class Diagram

A UML Class Diagram representing the structure of the Lost & Found Portal system.

* Includes classes, attributes, and methods
* Shows relationships such as inheritance and association

📂 Folder: `Lab3_Class_Diagrams`

---

## Lab 4 – Sequence Diagrams

Sequence diagrams for different real-world systems:

* ATM System
* Medical Clinic System
* Car Insurance System

These diagrams illustrate interactions between system components over time.

📂 Folder: `Lab4_Sequence_Diagrams`

---

## Lab 5 – GitHub Portfolio

This lab demonstrates how to organize and publish all course work in a GitHub repository.

---

## Lab 6 – TDD Bowling Game (Python)

Implementation of a bowling game scoring system using Test-Driven Development (TDD).

* Developed using Python
* Includes unit tests
* Covers strikes, spares, and full game scoring

📂 Folder: `Lab1_TDD`
